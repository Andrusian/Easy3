EASY3 Audio Generator

---This time in Android---
Note that this is a different project from EASY (command-line audio genrator written in Python) and EASY2 (command line audio generator written in C++).
This provide 4 primary carrier oscilators and 4 amplitude modulators.
It has a present store/restore system as well as a sequencer.
There are additional features such as periodic silence and boost options.
Phase can be controlled in the left carrier and in some amods.
Multiple waveform types and duty cycles for some waveforms are supported.
